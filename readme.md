caddy官方一键安装
```
# 确保80端口没有被其它程序占用才能安装caddy
curl https://getcaddy.com | bash
# 可以加-s参数安装插件 如安装文件柜插件: curl https://getcaddy.com | bash -s http.filemanager
```
